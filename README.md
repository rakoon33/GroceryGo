# GroceryGo

An iOS application for online grocery shopping.  
It helps users browse products, add items to the cart, make quick checkouts, and manage their orders with ease.

## Features
- User registration and login
- Browse products by categories
- Search and view product details
- Add products to cart and update quantities
- Checkout and place orders
- Manage account and view order history

## Requirements
- iOS 13.0 or later
- Xcode 15 or later
- Swift 5 or later

## Installation & Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/GroceryGo.git
   ```
2. Open the .xcodeproj or .xcworkspace file in Xcode
3. Select a simulator or a real device
4. Press Run to build and launch the app
